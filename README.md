# react-native-fast-drawer

React native drawer using Fabric architecture

## Installation

```sh
npm install react-native-fast-drawer
```

## Usage

```js
import { FastDrawerView } from "react-native-fast-drawer";

// ...

<FastDrawerView color="tomato" />
```
